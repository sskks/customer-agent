# V2 功能说明 - AI 脚本生成

## 🎉 新功能概述

V2 版本新增了 **AI 智能脚本生成** 功能，可以基于推荐的选题自动生成完整的短视频拍摄脚本，包括：

- ✅ **开头钩子**（前3秒抓住注意力）
- ✅ **完整口播文案**（150-250字）
- ✅ **行动号召**（引导用户互动）
- ✅ **分镜脚本**（3-5个场景的详细画面描述）
- ✅ **拍摄建议**（灯光、道具、角度等专业指导）

---

## 🚀 快速开始

### 1. 启动开发服务器

```bash
cd "D:\vscode\Microsoft VS Code\project\customer agent\dianbo-agent"
npm run dev
```

服务器将在 http://localhost:3000 启动

### 2. 使用流程

1. **打开浏览器** 访问 http://localhost:3000
2. **输入信息**：
   - 选择行业类型（美容院/餐饮店/健身房）
   - 输入店铺名称
3. **生成推荐**：点击"生成今日推荐"按钮
4. **生成脚本**：点击任意推荐的"生成脚本"按钮
5. **查看结果**：在弹窗中查看完整的拍摄脚本

### 3. 脚本功能

每个生成的脚本包含：

#### 📋 脚本概览
- 视频标题
- 预计时长（通常 30-60 秒）
- 分镜数量

#### 🎯 开头钩子
- 前3秒的吸睛台词
- 使用提问/反差/悬念/数据等技巧

#### 📝 完整口播文案
- 150-250字的主体内容
- 口语化表达，适合直接朗读

#### 📣 行动号召
- 最后5秒的转化引导
- 引导评论/私信/到店

#### 🎬 分镜脚本
每个分镜包含：
- 场景编号和时长
- 画面描述（镜头、场景、人物动作）
- 对应台词或音效
- 拍摄备注（道具、角度等）

#### 💡 拍摄建议
- 灯光布置建议
- 声音录制技巧
- 背景布置要点
- 其他专业提示

## 💰 成本说明

### 当前配置（演示模式）

**无需 API Key** - 系统使用模拟数据生成示例脚本
- 成本：0 元
- 用途：产品演示和功能测试

### 生产环境配置

如需使用真实的通义千问 API：

1. **注册阿里云账号**
   - 访问 https://dashscope.aliyun.com/
   - 创建账号并开通 DashScope 服务

2. **获取 API Key**
   - 在控制台创建 API Key
   - 复制 Key 备用

3. **配置环境变量**
   
   在项目根目录创建 `.env.local` 文件：
   ```env
   QWEN_API_KEY=your_api_key_here
   ```

4. **成本估算**
   - 模型：qwen-plus
   - 输入：0.008元/1K tokens
   - 输出：0.02元/1K tokens
   - **平均每次生成：约 0.03 元**

## 🛠️ 技术实现

### 核心文件

- **lib/scriptGenerator.ts** - 脚本生成服务
  - `generateScript()` - 主函数
  - `buildPrompt()` - Prompt 工程
  - `callQwenAPI()` - API 调用
  - `parseResponse()` - 响应解析

- **app/api/script/route.ts** - API 端点
  - POST /api/script
  - 接收：topic, contentType, industry, businessName
  - 返回：GeneratedScript

- **app/page.tsx** - 前端界面
  - ScriptModal 组件 - 脚本展示弹窗
  - 复制/下载功能

### Prompt 工程策略

针对不同类型的内容采用不同的结构：

| 内容类型 | 结构模板 |
|---------|---------|
| customer_case | 问题 → 解决方案 → 效果对比 → 客户反馈 |
| knowledge | 痛点引入 → 专业讲解 → 实用建议 → 总结 |
| environment_tour | 开场吸引 → 空间介绍 → 细节展示 → 邀请体验 |
| promotion | 优惠信息 → 价值塑造 → 限时紧迫感 → 行动号召 |
| behind_scenes | 日常场景 → 工作细节 → 用心之处 → 情感共鸣 |
| product_showcase | 产品亮相 → 核心卖点 → 使用演示 → 购买引导 |

### 行业语气指导

| 行业 | 语气风格 |
|-----|---------|
| beauty | 语气温和专业，突出美丽和自信，适合女性受众 |
| restaurant | 语气热情活泼，突出美食诱惑，适合年轻人群 |
| fitness | 语气激励鼓舞，突出健康和改变，适合追求进步的人群 |

## ❓ 常见问题

### Q1: 为什么生成的脚本是中文乱码？

A: 这是终端显示的编码问题，实际数据是正确的。在浏览器中查看会正常显示中文。

### Q2: 如何自定义 Prompt 模板？

A: 编辑 `lib/scriptGenerator.ts` 中的 `buildPrompt()` 函数，修改行业指导和内容结构模板。

### Q3: 可以更换其他 LLM 模型吗？

A: 可以。修改 `callQwenAPI()` 函数，替换为其他模型的 API 调用逻辑即可。

### Q4: 脚本生成的质量如何保证？

A: 
- 使用经过验证的 Prompt 模板
- 针对不同行业和内容类型定制语气
- 提供拍摄建议和专业指导
- 支持重新生成直到满意

### Q5: 生成的脚本可以直接使用吗？

A: 是的，生成的脚本已经是口语化表达，可以直接用于拍摄。但建议根据实际情况微调细节。
