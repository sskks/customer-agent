## 📋 准备工作

### 1️⃣ 购买腾讯云服务器

推荐配置（新用户约 50-100 元/月）：
- **实例类型**：标准型 S5/S6
- **配置**：2核 4GB（建议 4核 8GB）
- **带宽**：3Mbps 以上（建议 5Mbps+）
- **操作系统**：Ubuntu 22.04 LTS
- **地域**：上海/广州/北京（离你最近的节点）

购买地址：https://cloud.tencent.com/act/pro/cvm

### 2️⃣ 配置安全组

在腾讯云控制台 → 云服务器 → 安全组 → 添加入站规则：

| 端口 | 协议 | 来源 | 说明 |
|------|------|------|------|
| 22 | TCP | 0.0.0.0/0 | SSH 远程登录 |
| 3000 | TCP | 0.0.0.0/0 | 应用服务 |
| 80 | TCP | 0.0.0.0/0 | HTTP（可选） |
| 443 | TCP | 0.0.0.0/0 | HTTPS（可选） |

## 🚀 一键部署（5 分钟完成）

### 第一步：SSH 登录服务器

在本地电脑打开 PowerShell 或终端：

```powershell
ssh root@你的服务器IP
```

### 第二步：安装 Docker 和 Docker Compose

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Docker
apt install -y docker.io docker-compose

# 启动 Docker 并设置开机自启
systemctl start docker
systemctl enable docker

# 验证安装
docker --version
docker-compose --version
```

### 第三步：下载项目代码

```bash
cd /opt
git clone https://github.com/sskks/customer-agent.git dianbo-agent
cd dianbo-agent
```

### 第四步：配置环境变量

```bash
# 复制配置模板
cp .env.production.example .env

# 编辑配置文件（填入你的 API Key）
nano .env
```

需要填写的配置：
```ini
# 通义千问 API Key（必需）
QWEN_API_KEY=sk-your-api-key-here

# Supabase 配置（必需）
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
```

编辑完成后按 `Ctrl+O` 保存，`Ctrl+X` 退出。

### 第五步：启动应用

```bash
# 使用国内优化版构建（推荐，速度更快）
docker-compose -f docker-compose.cn.yml up -d --build

# 查看启动日志
docker-compose logs -f
```

看到 `ready started server on 0.0.0.0:3000` 表示启动成功！

按 `Ctrl+C` 退出日志查看。

## 🌐 进阶配置（可选）

### 配置域名访问

1. 在腾讯云购买域名（如 `dianbo.yourdomain.com`）
2. 域名解析到你的服务器 IP
3. 配置 Nginx 反向代理（项目已内置 nginx.conf 模板）

### 配置 HTTPS 证书

使用 Let's Encrypt 免费证书：

```bash
# 安装 certbot
apt install -y certbot python3-certbot-nginx

# 申请证书
certbot --nginx -d 你的域名
```

## ❓ 常见问题

### Q: 构建速度很慢怎么办？

A: 使用 `docker-compose.cn.yml`（已配置国内镜像源），比原版快 5-10 倍。

### Q: 服务器防火墙怎么配置？

A: 除了腾讯云控制台安全组，还需要检查服务器内部防火墙：

```bash
# Ubuntu UFW 防火墙
ufw allow 22/tcp
ufw allow 3000/tcp
ufw enable
```

### Q: 如何更新代码到最新版本？

```bash
cd /opt/dianbo-agent
git pull
docker-compose -f docker-compose.cn.yml up -d --build
```

### Q: 通义千问 API Key 怎么获取？

1. 访问：https://dashscope.console.aliyun.com/
2. 登录阿里云账号
3. 创建 API Key
4. 复制填入 .env 文件
